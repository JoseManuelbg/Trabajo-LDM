Definición

Las RSS (Really Simple Syndication) son un formato de distribución de contenido web que permite a los usuarios obtener actualizaciones de sitios web de manera conveniente. Funcionan mediante la publicación de contenido en un formato XML estructurado, que incluye resúmenes de noticias o artículos junto con enlaces al contenido completo. Los usuarios pueden suscribirse a los feeds RSS de los sitios web que deseen seguir utilizando un lector de feeds SS, también conocido como agregador de noticias. Este software recopila y muestra todas las actualizaciones de los sitios web a los que el usuario está suscrito en una sola interfaz.
